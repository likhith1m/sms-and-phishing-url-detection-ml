# SMS Spam & Phishing URL Detection

This project detects whether a message is spam and whether a URL is phishing using Machine Learning.

## Features
- SMS Spam Detection
- Phishing URL Detection
- Built with Machine Learning
- Streamlit Web Interface

## Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## How to Run

1. Clone the repository
git clone https://github.com/yourusername/repository-name

2. Install dependencies
pip install -r requirements.txt

3. Run the app
streamlit run app.py
